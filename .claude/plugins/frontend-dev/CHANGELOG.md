# Changelog - Frontend Dev Plugin

All notable changes to this plugin will be documented in this file.

## [1.0.0] - 2026-01-21

### 🎉 Initial Release

Complete frontend development toolkit for React + TypeScript projects.

### ✨ Features Added

#### Agents (4)
- **code-reviewer** - React/TypeScript code quality and best practices
- **accessibility-analyzer** - WCAG 2.1 AA compliance auditing
- **test-coverage-analyzer** - Test gap analysis and recommendations
- **performance-analyzer** - Bundle size and runtime optimization

#### Commands (6)
- **generate** - Scaffold components, hooks, utilities, pages with tests
- **build** - Production build with bundle analysis
- **test** - Unit testing with Vitest (watch, run, coverage, ui modes)
- **lint** - ESLint code quality checks with auto-fix
- **e2e** - Playwright E2E testing across browsers
- **feature-dev** - Guided 9-stage feature development workflow

#### Skills (4)
- **react-patterns** - Modern React 19 patterns and hooks
- **testing-strategy** - Comprehensive testing guide (unit, integration, E2E)
- **mui-customization** - Material UI v7 theming and styling
- **zustand-patterns** - State management with Zustand

### 📚 Documentation
- README.md - Complete plugin documentation
- QUICK_REFERENCE.md - Developer quick reference
- EXAMPLES.md - Real-world usage examples
- CHANGELOG.md - This file

### 🏥 Healthcare Features
- HIPAA compliance reminders in generated code
- PHI handling best practices
- Security-aware templates

### 🎯 Quality Standards
- Test coverage: ≥80% (unit), ≥85% (critical)
- Bundle size: <200KB gzipped
- Accessibility: WCAG 2.1 AA compliant
- TypeScript strict mode enforced
- ESLint zero errors policy

### 📦 Tech Stack Support
- React 19.1 with TypeScript 5.9
- Vite 7 (build tool)
- Material UI v7
- Zustand 5 (state management)
- Vitest 4 (unit testing)
- Playwright 1.57 (E2E testing)
- React Testing Library
- ESLint 9
