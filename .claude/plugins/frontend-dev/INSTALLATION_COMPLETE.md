## 🚀 Quick Start

### 1. Try Your First Command

```bash
# Generate a component
/frontend-dev:generate component HelloWorld

# Run tests
/frontend-dev:test

# Build for production
/frontend-dev:build
```

### 2. Use an Agent

Agents run automatically on relevant operations:
- **Code reviewer** → Analyzes git diff in PRs
- **Accessibility analyzer** → Checks WCAG compliance
- **Test coverage analyzer** → Finds coverage gaps
- **Performance analyzer** → Optimizes bundle size

### 3. Reference a Skill

Ask Claude:
- "Show me React patterns"
- "How do I test a custom hook?"
- "What's the best way to customize MUI?"
