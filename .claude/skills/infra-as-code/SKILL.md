# Infra as Code Skill

## When This Skill Activates
- Provisioning cloud/on-prem infrastructure
- Generating Terraform or Ansible templates
- Creating environment-specific infrastructure plans

## Capabilities
- Terraform module scaffolding
- Ansible playbook generation
- Environment parameterization
- Idempotent provisioning checks

## Usage
```bash
Skill("infra-as-code")
```
