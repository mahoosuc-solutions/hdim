## Key Architecture Decisions

| Decision | Impact | Rationale |
|----------|--------|-----------|
| **3-Portal Structure** | Audience segmentation | Different users need different content |
| **Metadata-Driven** | Searchability & flexibility | 40+ fields enable rich queries |
| **Template-Based** | Consistency & speed | Writers copy templates, fill content |
| **Git-Based Storage** | Version control & audit | Markdown in git, metadata in DB |
| **Elasticsearch** | Fast search at scale | Required for 115+ documents, 600+ keywords |
| **4 Workflows** | Handles all change types | Different triggers need different approvals |
| **Owner-Per-Doc** | Clear accountability | 115 documents, 115 owners |
| **QA Automation** | Consistent quality | Automated checks for SEO, accessibility, links |
