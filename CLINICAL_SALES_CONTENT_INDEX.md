## How These Assets Work Together

### The Sales Motion

```
PROSPECT IDENTIFIED
        ↓
[SEND] One-pager + Email #1 (CLINICAL_OVERVIEW_ONEPAGER.md + EMAIL_NURTURE_SEQUENCE.md #1)
        ↓
[IF INTERESTED] Discovery call (SALES_QUICK_REFERENCE.md discovery questions)
        ↓
[SEND] Case study + Email #2-3 (CASE_STUDY_CLINICAL_IMPACT.md + EMAIL_NURTURE_SEQUENCE.md #2-3)
        ↓
[SCHEDULE] Working session
        ↓
[IN SESSION] Run ROI calculator + demo (roi-calculator.ts + CLINICAL_OVERVIEW_ONEPAGER.md)
        ↓
[SEND] Proposal with customized ROI (CLINICAL_SALES_STRATEGY.md financial model)
        ↓
[CLOSE] Deal
```

## FAQ: Using This Content Suite

**Q: Which document should I start with?**
A: **Sales team:** Start with SALES_QUICK_REFERENCE.md (30 seconds of daily learning)  
**Marketing team:** Start with CLINICAL_SALES_IMPLEMENTATION_SUMMARY.md (deployment guide)  
**Executives:** Start with CLINICAL_SALES_STRATEGY.md Section 2 (ROI model)

**Q: Can I modify these documents?**
A: Yes. They're templates designed for customization. Add your branding, adjust ROI assumptions, personalize examples. Just maintain the structure.

**Q: What if my organization has different ROI?**
A: The ROI model shows assumptions in CLINICAL_SALES_STRATEGY.md Section 2.2. Adjust based on your market, customer size, implementation approach. Use roi-calculator.ts to model different scenarios.

**Q: How do I measure success?**
A: Track metrics in each section. Monitor email open/click rates, demo booking rates, proposal advance rates, close rates. Compare to targets in "Success Metrics" section above.

**Q: Can I combine these into a single document?**
A: Yes, but I recommend keeping them separate. They serve different audiences and use cases. Marketing automation works better with separate email files. Prospects prefer scannable one-pager to 8K strategy doc.

**Q: What's the best way to introduce this to the team?**
A: 1) Share CLINICAL_SALES_IMPLEMENTATION_SUMMARY.md with everyone. 2) Do 30-minute training on SALES_QUICK_REFERENCE.md with sales team. 3) Set up email automation with EMAIL_NURTURE_SEQUENCE.md. 4) Brief marketing on ROI calculator deployment.
