# Cost Attribution Model

## Objective

Attribute infrastructure and runtime cost to service, feature, and tenant dimensions.

## Dimensions

- Service
- Feature or workflow
- Tenant
- Time window (hour/day/week/month)

## Cost Inputs

- CPU usage
- Memory usage
- Storage and I/O
- Network egress

## Aggregation

- Raw event-level tracking for detailed attribution
- Daily summaries for dashboard speed
- Historical trend retention for planning
