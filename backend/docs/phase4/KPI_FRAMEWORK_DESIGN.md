# KPI Framework Design

## Objective

Provide real-time and trend KPI visibility for clinical outcomes and operational performance.

## Clinical KPIs

- Measure evaluations
- Care gap detection and closure rates
- Time to closure

## Business/Operational KPIs

- Availability and latency SLOs
- Error rates by service
- Cost-per-workflow and trend

## Delivery

- Dashboard-ready aggregates
- API outputs for reporting workflows
- Threshold-based alerts for KPI regressions
