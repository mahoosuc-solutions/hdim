**For Due Diligence (Provide VCs Access):**
- [ ] GitHub repository (read-only access)
- [ ] Financial projections (spreadsheet)
- [ ] Cap table (current share ownership)
- [ ] Customer agreements (with privacy)
- [ ] HIPAA compliance documentation
- [ ] Regulatory clearances (if any)
- [ ] IP documentation (patents, trademarks, copyrights)
- [ ] Employment agreements with key team
- [ ] Insurance policies (general liability, D&O)

### Step 3: Negotiate Term Sheet
**Key Terms to Negotiate:**
- [ ] Valuation (expect $15-25M pre-money for Series A)
- [ ] Investment amount ($5-7M)
- [ ] Board seat (1 investor, founder, 1 independent)
- [ ] Liquidation preference (1x non-participating preferred standard)
- [ ] Anti-dilution (weighted average preferred)
- [ ] Information rights (quarterly updates)
- [ ] Pro-rata rights (follow-on investment opportunity)
- [ ] Drag-along rights (exit decision making)

**Red Flags to Avoid:**
- [ ] 2x liquidation preference (too aggressive)
- [ ] Full ratchet anti-dilution (punishes down rounds)
- [ ] 3+ investor board seats (limits founder control)
- [ ] Redemption rights (unlikely but check)
- [ ] Excessive information rights (time-consuming)

### Step 4: Legal Review
- [ ] Engage startup lawyer (costs: $3-10K for Series A)
- [ ] Have lawyer review term sheet before signing
- [ ] Prepare shareholder list for due diligence
- [ ] Ensure proper stock option grants to employees
- [ ] Verify 83(b) elections filed for all early employees
