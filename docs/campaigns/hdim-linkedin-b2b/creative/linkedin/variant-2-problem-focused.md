### Style Guidelines

- **Color Palette**:
  - Primary: HDIM brand blue (#0A66C2)
  - Alert/Problem: Warning red (#E74C3C) for spaghetti lines, costs, X icons
  - Accent: Neutral gray (#95A5A6) for background elements

## Designer Brief

**Objective**: Create a problem-agitation LinkedIn ad that resonates with frustrated healthcare IT leaders dealing with integration hell. The ad should make them say "This is EXACTLY my problem."

**Target Audience**: CIOs and IT Directors at hospitals who are drowning in integration maintenance and feel like they're constantly firefighting instead of innovating.

**Key Message**: Current EHR integration approaches are expensive, fragile, and unsustainable. There's a better way.

**Emotional Tone**:
- 70% Problem/Pain (empathy + agitation)
- 30% Hope/Solution (there's a better way)
- NOT: Fear-mongering or overly negative

**Design Requirements**:
1. **Visual Metaphor**: The "spaghetti diagram" must be instantly recognizable as integration complexity
2. **Cost Shock**: The $500K figure should jump out (large, bold, red)
3. **Authenticity**: Use real EHR vendor logos (Epic, Cerner) for credibility - healthcare IT buyers know these names
4. **Problem-Solution Contrast**: Upper portion shows chaos, lower portion shows clean CTA (solution)
5. **Mobile Readability**: Text must be legible at thumbnail size

**Brand Guidelines**:
- HDIM positioning: The "anti-complexity" solution
- Tone: Technical, empathetic, no-nonsense (NOT salesy)
- Avoid: Over-simplified graphics (audience is technical)

**Deliverables**:
1. Primary ad: 1200×627px PNG (Layout Option 1 - Spaghetti)
2. A/B test variant: 1200×627px PNG (Layout Option 2 - Cost Breakdown)
3. Square variants: 1080×1080px PNG for both layouts
4. Editable source files

**Timeline**: 5-7 business days (more complex than Variant 1)

## Lead Gen Form Customization

Since this variant uses "Free Assessment" CTA, customize the Lead Gen Form:

**Form Fields** (8 total):
1. First Name (pre-filled)
2. Last Name (pre-filled)
3. Work Email (pre-filled)
4. Company (pre-filled)
5. Job Title (pre-filled)
6. Hospital Bed Count (dropdown: <200, 200-500, 500-1000, 1000+)
7. Number of EHR Systems (dropdown: 1, 2, 3, 4+)
8. Current Annual Integration Budget (dropdown: <$100K, $100-250K, $250-500K, $500K+, Don't know)

**Why These Fields**:
- Fields 6-8 allow immediate assessment calculation
- Pre-qualification: Hospitals with multiple EHRs and budgets >$250K are higher intent
- Gives sales team context before first call

**Potential Issue**: 8 fields may reduce completion rate below 40%
- **Mitigation**: Make fields 6-8 optional, or use progressive profiling (ask in follow-up email)
