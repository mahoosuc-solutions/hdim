# HDIM Financial Forecast (Current Structure)

**Date:** 2026-03-07  
**Entity:** Grateful House Incorporated  
**Model:** Source-available (BSL 1.1) + commercial production licenses + implementation + support

This forecast is an operating planning model, not investment advice.

## 1) Current Structure Assumptions

- Product remains source-available under BSL 1.1.
- Commercial revenue comes from production license subscriptions, implementation fees, and support/compliance services.
- Initial GTM uses founder-led sales + HIMSS pipeline.
- Delivery model remains lean (small core team, selective contractors).
- Forecast starts in **Q2 2026** (post-HIMSS execution window).

## 7) Cash Planning and Runway Signals (Base)

- Peak annual burn expected during 2027 scale year.
- Minimum cash threshold target: 9-12 months forward runway.
- Trigger points for course correction:
  - If Q3 2026 bookings < 60% of plan: reduce hiring and focus on services revenue.
  - If net revenue retention < 90% in 2027: pause expansion and fix retention.
  - If CAC payback exceeds 18 months: tighten ICP and partner-led channels.

## 8) Leading Indicators to Track Monthly

- Qualified pipeline value (3x-4x next 2 quarters target bookings)
- Win rate on qualified opportunities
- Sales cycle length (target < 120 days for initial commercial deals)
- Net revenue retention (target >= 100% by 2028)
- Gross margin by revenue stream
- Burn multiple (net burn / net new ARR)

## 9) Execution Priorities to Support Forecast

1. Convert HIMSS activity into 2-3 paid pilots within 90 days.
2. Standardize a 6-week implementation package to protect services margin.
3. Productize compliance/support tiers for predictable expansion revenue.
4. Establish quarterly pricing governance and discount approval thresholds.
5. Use source-available funnel metrics (stars, inbound demos, POCs) as top-of-funnel inputs.
