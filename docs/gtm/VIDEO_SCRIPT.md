## Production Notes

### Screen Capture Requirements
- Use demo environment with sample data
- Ensure PHI-free sample patients
- 60fps screen recording for smooth cursor movement
- Zoom on key interface elements

### Voiceover Direction
- Professional but warm tone
- Conversational, not corporate
- Emphasize key statistics with slight pause
- 150 words per minute pace

### Music
- Modern, upbeat but professional
- Fade under voiceover, up during transitions
- No lyrics

### Graphics
- Animated statistics at 4:00
- Subtle callout boxes for key features
- HDIM brand colors throughout
- Lower thirds for section titles
