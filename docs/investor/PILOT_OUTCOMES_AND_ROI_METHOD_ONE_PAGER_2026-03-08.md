# Pilot Outcomes and ROI Method (One-Pager)

**Date:** 2026-03-08  
**Objective:** Provide an auditable method for pilot outcome and ROI claims.

## Outcome Model
Pilot performance is measured on a single scorecard with baseline, target, and current value for each KPI:
- Care-gap closure lift
- Time-to-intervention
- Denied-claim reduction
- Manual-work reduction
- Platform availability and p95 latency
- Incident rate

Reference template: `docs/templates/PILOT_SCORECARD_TEMPLATE.md`

## ROI Calculation Method
ROI is computed from four components with explicit lineage:
1. Incremental clinical/quality impact value
2. Denial reduction and reimbursement uplift
3. Labor time saved from workflow automation
4. Avoided penalty / incentive improvement

Each component must include:
- Baseline window
- Measurement window
- Formula and assumptions
- Confidence level
- Source artifacts/queries

Reference template: `docs/investor/ROI_DEFENSIBILITY_PACK_TEMPLATE.md`

## Evidence Standards
- No unsupported headline claims.
- Every figure maps to a reproducible source.
- Confidence band required for executive summary.
- Caveats and exclusions explicitly listed.

## Decision Use
- **Customer:** validate expected operational and clinical benefit before go-live.
- **Investor:** validate repeatable value creation with defensible methodology.
