### Calculation Notes

**CAC (Customer Acquisition Cost)**
```
CAC = (Sales & Marketing Spend) / (New Customers)
Year 1: ($120K + $150K sales salary) / 6 net new = $45K
```

**LTV (Lifetime Value)**
```
LTV = (ARPU × Gross Margin) / Churn Rate
Year 1: ($60K × 75%) / 25% assumed = $180K
```

**Magic Number**
```
Magic Number = (Net New ARR) / (Prior Quarter S&M Spend)
Year 3 Q4: ($1.2M net new) / ($150K S&M) = 0.8
```
