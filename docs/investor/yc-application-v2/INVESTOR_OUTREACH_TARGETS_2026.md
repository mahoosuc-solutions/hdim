## Outreach message angles by firm type

- Healthcare specialist funds:
  - Emphasize quality and interoperability wedge, deployment speed, and mid-market ACV fit.
- Platform crossover funds:
  - Emphasize AI-native engineering economics, architecture moat, and scalable product surface.
- Strategic funds:
  - Emphasize pilot readiness, measurable workflow outcomes, and integration roadmap.

## 4-week outreach execution plan

Week 1:
- Build partner/principal target list per firm (3-5 contacts each).
- Prepare 1-page investor brief + 10-slide deck + product walkthrough.

Week 2:
- Send 10-15 warm intro requests through operators/founders/advisors.
- Run first-call scripts tailored by investor type.

Week 3:
- Run 6-10 first meetings.
- Track objections in a shared objection log and update collateral.

Week 4:
- Prioritize top 4 firms for partner meetings.
- Package data room with architecture docs, security artifacts, and customer proof points.

## Data room checklist for this outreach

- YC application narrative (current)
- Financial model + unit economics assumptions
- Product architecture one-pager and technical deep dive
- Security/compliance artifacts (HIPAA/SOC2 progress)
- Demo scripts and release evidence
- Customer pipeline + pilot conversion assumptions
