# Story QA Checklist

Use this checklist before finalizing any sales narrative update.

## Core Checks

- [ ] Claim-to-proof link: Every major claim has a nearby evidence link.
- [ ] Date specificity: Time-based statements include concrete dates (example: November 24, 2024).
- [ ] Persona CTA: Each buyer persona section includes a clear next step or asset to review.
- [ ] Metric consistency: Metrics are numerically and contextually consistent across all linked collateral.
