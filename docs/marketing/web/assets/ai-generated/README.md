# AI-Generated Portal Assets

This directory contains AI-generated visual assets for the HDIM AGUI portal.

## Governance Requirements
- Every asset must have a provenance record in `provenance-ledger.json`.
- PHI content is prohibited in all prompts and outputs.
- Publication requires human approval by compliance and security owners.
- Generated assets are treated as release evidence inputs and retained with release artifacts.

## Current Asset Set
- `platform-control-plane.svg`
- `persona-map.svg`
- `compliance-loop.svg`
- `event-fabric.svg`

## Validation
Reviewers should verify:
1. No PHI or tenant-identifying details in visual text.
2. Prompt summary and model metadata exist.
3. Approval fields are complete.
