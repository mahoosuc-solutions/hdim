## 3. Hashtag Strategy

Add these as a **comment** on the existing post (not an edit to the original post — LinkedIn penalizes edits):

```
#HEDIS #QualityMeasures #ValueBasedCare #MedicareStars #CareGaps
```

**Rationale:**
| Hashtag | Why |
|---------|-----|
| `#HEDIS` | Core topic — reaches HEDIS practitioners directly |
| `#QualityMeasures` | Broader quality measurement audience beyond HEDIS — covers MIPS, Stars, HEDIS |
| `#ValueBasedCare` | Reaches VBC executives, ACO leaders, payer strategy teams |
| `#MedicareStars` | Reaches MA plan quality teams; HEDIS measures are ~40% of Star ratings score |
| `#CareGaps` | Specific clinical term — reaches quality improvement and care management teams |

**Optional secondary hashtags** (use in group posts, not on original):
- `#MedicareAdvantage` — if sharing in MA/Stars groups
- `#MIPS` — if sharing in MIPS/QPP groups (provider-side quality reporting)
- `#FHIR` — if sharing in interoperability groups
- `#PopulationHealth` — if sharing in population health groups
- `#StarRatings` — if sharing in CMS Star ratings groups
- `#QPP` — if sharing in Quality Payment Program / MIPS groups

## 5. Engagement Rules

1. **Never mention HDIM or that you're building something** — same rule as Tier 1 comments
2. **Add genuine insight, not validation** — every comment should teach something
3. **2-3 group shares per day max** — LinkedIn throttles aggressive group posting
4. **Stagger engagement** — don't like/comment on 10 things in 5 minutes
5. **Wait 24-48 hours after joining a group** before posting
6. **Track what works** — note which groups/people drive profile views back to your post
