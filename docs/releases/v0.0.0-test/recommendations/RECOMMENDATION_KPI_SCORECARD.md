# Recommendation: KPI Scorecard

## What We Are Building
A single pilot scorecard that ties clinical, operational, and technical KPIs to owners and cadence.

## How To Implement
- Use `docs/releases/v0.0.0-test/validation/pilot-scorecard.md` as source of truth.
- Update weekly from validated system data.
- Trigger review for >10% KPI regression.

## Expected Outcomes
- Shared KPI language across executives and operators.
- Faster GO/NO-GO decisions.
- Auditable pilot performance trendline.
