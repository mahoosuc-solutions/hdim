# Handoff Matrix - Large Health System Pilot

## Handoff Table
| From Function | To Function | Trigger | Input Artifact | Output Artifact | SLA |
|---|---|---|---|---|---|
| Sales | Solution Architecture | Discovery call complete | Customer response package + discovery notes | Architecture assumptions + decision agenda | 1 business day |
| Solution Architecture | Engineering | Topology and controls selected | Deployment decision + control requirements | Integration plan + technical task list | 1 business day |
| Engineering | Customer Success | UAT scope defined | Integration readiness checklist + test plan | Pilot readiness checklist + comms plan | 1 business day |
| Customer Success | Executive Sponsors | Weekly steering complete | Weekly KPI + risk + action updates | Decision log + escalation requests | 1 business day |
| Security/Compliance | Sales + SA | Security review complete | Control checklist + exceptions | Contract/security terms confirmation | 2 business days |

## Escalation Rule
- If SLA is missed, escalate to Sales Lead and Solution Architect within 24 hours.
