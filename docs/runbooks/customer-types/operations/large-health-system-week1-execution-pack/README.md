## Immediate Usage Sequence

1. Confirm names and contacts in owner roster.
2. Use gate status snapshot in Week 1 architecture workshop.
3. Track all decisions/actions in action log during and after the workshop.
4. Update risk register at end of workshop.
5. Publish weekly steering update.
6. Complete G1 evidence and issue readiness memo.
7. Run internal Day 1 launch check and send workshop pre-read.
8. Use `17-todays-priority-todo.md` as daily command list until G1 passes.
9. Use `18-open-items-resolution-log.md` as the source of truth for unresolved items.
10. Execute outbound communications using `19-customer-communications-sequence.md`.
11. Use `20-day1-send-set.md` for copy/send-ready communication templates.
12. Use `21-day1-decision-packet-checklist.md` before sending Day 1 materials.
13. Track all external comms in `22-outbound-communications-log.md`.
14. Execute cadence in `23-week1-calendar-plan.md`.
15. Use `24-go-no-go-dashboard-week1.md` for end-of-week decision.
16. Use `25-week1-execution-summary-one-pager.md` as daily executive control sheet.
17. Run `26-owner-briefing-script.md` at start of each day.
18. Use `27-close-open-items-checklist.md` to close and verify unresolved items.
19. Use `28-week2-transition-plan.md` once Week 1 exit criteria are met.
20. Use `29-next-account-replication-checklist.md` to clone this system for the next account.
21. Use `30-g1-closure-worksheet.md` to force G1 evidence completion and decision.
22. Run `31-placeholder-resolution-checklist.md` before sending any external communication.
23. Complete `32-enterprise-security-addendum.md` for enterprise security sign-off.
24. Complete `33-onprem-change-control-addendum.md` for on-prem change governance (if applicable).
25. Fill `00-account-variables.md` first to speed placeholder resolution.
26. Review `34-placeholder-scan-report.md` and clear high-priority placeholders.
27. Track overall status in `35-current-tasks-completion-board.md`.
28. Use `36-account-variables-fill-now.md` for a guided one-pass completion of all human-required fields.
