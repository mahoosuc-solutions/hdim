# Customer Communications Sequence (Week 1)

Purpose: standardize customer communication flow from invite through decision confirmation.

## Sequence

1. Send workshop invite
- Template: `14-workshop-invite-email-ready.md`
- Owner: Customer Success Lead
- Timing: Day 1

2. Send pre-read package
- Template: `10-workshop-pre-read-email.md`
- Owner: Solution Architect
- Timing: 24h before workshop

3. Run workshop and capture decisions
- Templates: `09-workshop-agenda.md`, `12-workshop-notes-template.md`, `11-decision-log.md`
- Owner: Solution Architect
- Timing: Workshop day

4. Send post-workshop summary
- Template: `15-post-workshop-followup-email.md`
- Owner: Sales Lead or Customer Success Lead
- Timing: within 24h after workshop

5. Send weekly steering kickoff note
- Template: `06-weekly-steering-update-week1.md`
- Owner: Customer Success Lead
- Timing: end of Week 1

## Quality Checks Before Sending

- [ ] Correct customer names and roles inserted
- [ ] Dates/times and owners filled in
- [ ] Deployment model references consistent across documents
- [ ] Attachments/links verified
- [ ] Action deadlines align with action log
