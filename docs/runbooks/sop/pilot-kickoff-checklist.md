# SOP: Pilot Kickoff Readiness Checklist

## Objective
Gate kickoff on readiness evidence.

## Hard Gate Rule
No pilot kickoff until all required artifacts are signed off.

## Checklist
- [ ] Deployment model decision complete (Week 1)
- [ ] G1 architecture/security gate passed
- [ ] G2 integration/UAT gate passed
- [ ] KPI baseline documented
- [ ] Weekly steering cadence scheduled
- [ ] Escalation and support model agreed

## Outputs
- Kickoff approval record
