# SOP: Security Readiness Checklist

## Objective
Verify security controls and policy alignment before go-live.

## Checklist
- [ ] Deployment model selected (hosted/cloud/on-prem)
- [ ] Shared responsibility matrix approved
- [ ] Tenant isolation controls verified
- [ ] Encryption and key management approach approved
- [ ] Audit logging coverage validated
- [ ] Incident response contacts and workflow confirmed

## Outputs
- Security checklist sign-off
- Gate evidence package
