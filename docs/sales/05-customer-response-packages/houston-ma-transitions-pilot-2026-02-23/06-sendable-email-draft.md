Subject: Phase 1 Pilot Plan and Budget for Houston Transitions-of-Care Program

Dr. [Last Name],

Thank you again for the discussion and for sharing the prototype concept. We reviewed your requirements and aligned them to a concrete Phase 1 pilot structure.

We are aligned on the core principles:

1. Zero durable PHI storage in the AI layer and strict data minimization.
2. Plain-language, context-aware patient guidance after discharge.
3. Secure escalation path to clinicians now, with MyChart write-back in a later phase.
4. A measurable 90-day pilot focused on transitions-of-care outcomes.

We propose a 90-day pilot for 200-500 discharges with the following deliverables:

1. Secure post-discharge patient engagement workflow.
2. AI-generated layman summaries and guided Q&A.
3. Clinical escalation routing (in-platform and HIPAA-secure email alert option).
4. KPI tracking and weekly reporting on engagement/comprehension/response metrics.
5. Security and audit baseline suitable for MA/ACO stakeholders.

Deployment options for Phase 1:

1. HDIM-hosted (fastest pilot launch).
2. Customer-hosted cloud (customer account ownership).
3. On-prem/private data center, including air-gapped variant.

Estimated pilot budget:

1. HDIM-hosted base estimate: $277,000-$383,000 total.
2. Customer-hosted cloud: add approximately $25,000-$70,000.
3. On-prem/private data center: add approximately $80,000-$210,000.
4. Air-gapped on-prem: add approximately $40,000-$120,000 beyond on-prem.
