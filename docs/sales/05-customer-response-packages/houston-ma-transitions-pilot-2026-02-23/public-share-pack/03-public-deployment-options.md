# Public Deployment Options One-Pager

## Supported Models

### 1) Vendor-Hosted

Best for faster pilot launch and lower internal infrastructure overhead.

### 2) Customer-Hosted Cloud

Best for organizations requiring account-level cloud ownership while retaining managed application operations.

### 3) On-Prem / Private Data Center

Best for stricter locality and network control requirements. Air-gapped mode is available for highly restricted environments.

## Common Security Posture Across Models

1. Minimum-necessary data handling.
2. Request-scoped AI context.
3. Role-based access and auditability.

## Typical Tradeoffs

1. Faster launch: vendor-hosted.
2. Greater customer control: customer-cloud/on-prem.
3. Higher complexity and longer lead time: on-prem/air-gapped.

## Commercial Framing (Public)

1. Hosted pilot baseline: $277,000-$383,000 total for 90 days.
2. Customer-cloud and on-prem models add implementation and operations overhead based on control requirements.

## Recommended Decision Process

Use a Week-1 architecture workshop to select model and finalize timeline/commercial assumptions.
