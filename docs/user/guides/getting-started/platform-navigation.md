## Top Navigation Bar

The **Top Navigation Bar** appears on every screen and gives you quick access to key functions.

### Left Section: Branding & Logo

**HealthData Logo** (left-most)
- Click to go back to main Dashboard from anywhere
- Useful when you're deep in a section and want to reset

### Middle Section: Search

**Universal Search Bar**
- Search for: patients, care gaps, reports, features
- Type patient name, MRN, ID, or feature name
- Results appear instantly below
- Click result to navigate there

**Search Tips**:
- **Patient**: Type name or medical record number (MRN)
- **Feature**: Type "care gap" or "reports"
- **Shortcut**: Press `Ctrl+/` (Windows) or `Cmd+/` (Mac) to focus search from anywhere

### Right Section: Tools & Profile

**Notifications Bell** 🔔
- Shows count of unread notifications
- Click to see recent alerts
- Color indicates urgency:
  - 🔴 Red: Critical/urgent items
  - 🟡 Yellow: Important items
  - ⚪ Gray: General notifications

**Profile Icon** (your initials or photo)
- Click to reveal profile menu:
  - View Profile
  - Account Settings
  - Change Password
  - My Preferences
  - Sign Out

## Main Content Area

The **Main Content Area** (right side) shows the actual page content.

### Common Page Elements

**Page Header**
- Page title (e.g., "Patient Dashboard")
- Breadcrumb navigation (shows where you are)
- Action buttons (specific to page)

**Content Sections**
- Information cards/panels
- Data tables
- Charts and visualizations
- Forms and input fields

**Page Footer**
- Pagination (if applicable)
- Help link
- Last updated timestamp

### Breadcrumb Navigation

Shows your location in the platform hierarchy:
- **Example**: Dashboard > Patients > John Smith > Care Gaps
- Click any breadcrumb to go back to that level
- Useful for understanding where you are

## Finding Features by Role

### For Physicians

**View Your Patients**:
1. Left menu → **Patients**
2. See your patient list
3. Click patient to see full dashboard

**Manage Care Gaps**:
1. Left menu → **Care Management** → **Care Gaps**
2. See gaps assigned to you
3. Click gap to see details and take action

**Check Quality Metrics**:
1. Left menu → **Quality Metrics**
2. See your performance vs. targets
3. View improvement opportunities

**View Patient Communication**:
1. Open patient → **Communication** tab
2. See messages, encounter notes, results
3. Review next appointment

### For Care Managers

**View Your Patient List**:
1. Left menu → **Patients**
2. Filter by your team
3. See risk scores and key metrics

**Create Care Plan**:
1. Open patient → **Care Plans**
2. Click **"+ New Care Plan"**
3. Fill in goals and action items
4. Assign to team members

**Track Outreach**:
1. Left menu → **Care Management** → **Outreach**
2. See call/message log
3. Log new contact
4. Schedule follow-up

**View Risk Assessment**:
1. Open patient → **Risk Assessment**
2. See risk score and drivers
3. View recommended interventions

### For Clinical Staff

**View Tasks**:
1. Left menu → **My Tasks**
2. Filter by priority or patient
3. Click task to take action

**Document Care**:
1. Open patient → **Documentation**
2. Click **"+ New Entry"**
3. Use templates or free text
4. Save and submit

**Send Communication**:
1. Open patient → **Communication**
2. Click **"Send Message"** or **"Schedule Call"**
3. Fill in details
4. Send or schedule

**Check Escalations**:
1. Left menu → **Escalations**
2. See flagged patients/gaps
3. Review clinical concerns
4. Escalate to provider if needed

### For Administrators

**View Reports**:
1. Left menu → **Reporting**
2. Choose report type
3. Set filters and date range
4. Generate or schedule

**Manage Users**:
1. Left menu → **Administration** → **User Management**
2. Search for user or add new
3. Assign roles and permissions
4. Deactivate when needed

**Monitor Data Quality**:
1. Left menu → **Administration** → **Data Quality**
2. See data completeness metrics
3. Identify data gaps
4. Take corrective action

**Review Audit Logs**:
1. Left menu → **Administration** → **Audit Logs**
2. Filter by user, date, or action
3. View who accessed what when
4. Export for compliance

### For IT Support

**Check System Health**:
1. Left menu → **Administration** → **System Status**
2. See system health metrics
3. View performance graphs
4. Check integration status

**Manage Integrations**:
1. Left menu → **Administration** → **Integrations**
2. See integration status
3. View recent sync history
4. Trigger manual sync if needed

**Monitor Backups**:
1. Left menu → **Administration** → **Backups**
2. See last backup timestamp
3. View backup history
4. Verify backup integrity

**View Audit Trail**:
1. Left menu → **Administration** → **Audit Logs**
2. Filter by user or action
3. Export logs for review
4. Monitor security events

## Search Functionality

### Global Search

**Activate Search**:
- Click search bar at top (magnifying glass 🔍)
- Or press `Ctrl+/` (Windows) or `Cmd+/` (Mac)

**Search Types**:

**Patient Search**:
- Type patient name: "John Smith"
- Type MRN: "MR12345"
- Type ID: "12345"
- Results show patient card with key info

**Care Gap Search**:
- Type "gap: diabetes" or just "diabetes"
- Results show matching gaps
- Filter by status or priority

**Feature Search**:
- Type "care plan" or "reports"
- Results show links to that feature
- Helps when you can't find a feature

**Report Search**:
- Type report name: "Quality Summary"
- Results show available reports
- Click to generate or schedule

### Advanced Search (Role-Specific)

Some features have **Advanced Search** options:
1. Click **"Advanced Search"** link below search bar
2. Set multiple filters
3. Choose data range
4. View results in table
5. Export if needed

## Keyboard Shortcuts

**Navigation Shortcuts**:
- `Home`: Go to Dashboard
- `Ctrl + /` (Windows) or `Cmd + /` (Mac): Open Search
- `Esc`: Close menus/dialogs
- `Tab`: Move to next item (navigation)
- `Shift + Tab`: Move to previous item

**See All Shortcuts**:
1. Click **Help** (bottom left)
2. Select **"Keyboard Shortcuts"**
3. View complete list for your role

## Troubleshooting Navigation

### I Can't Find a Feature

**Try These Steps**:
1. Use **Global Search** (Ctrl+/)
2. Type the feature name
3. Click result to go there
4. Or contact support

### A Menu Item Is Missing

**Possible Reasons**:
- Not available for your role
- Not enabled for your organization
- You don't have permission

**Solution**:
- Contact your administrator
- Request access if needed
- Check your role in Settings

### Page Doesn't Load

**Try These**:
1. Refresh browser (Ctrl+R or Cmd+R)
2. Clear browser cache
3. Try in private/incognito mode
4. Check internet connection
5. Try different browser
6. Contact IT support
