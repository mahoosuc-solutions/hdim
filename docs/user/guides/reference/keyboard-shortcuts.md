# Keyboard Shortcuts & Pro Tips

Work faster with shortcuts and hidden features.

## Browser Shortcuts (All Systems)

| Shortcut | Action |
|----------|--------|
| Ctrl+R (Cmd+R on Mac) | Refresh page |
| Ctrl+Shift+R (Cmd+Shift+R) | Hard refresh (clear cache) |
| Ctrl+F (Cmd+F) | Find text on page |
| Ctrl+L (Cmd+L) | Jump to address bar |
| Ctrl+T (Cmd+T) | New tab |
| Ctrl+W (Cmd+W) | Close tab |
| Ctrl+Shift+T (Cmd+Shift+T) | Reopen closed tab |
| Ctrl+Tab | Switch to next tab |
| Ctrl+Shift+Tab | Switch to previous tab |

## Platform Navigation Shortcuts

| Shortcut | Action |
|----------|--------|
| / | Focus search box |
| P | Jump to patient panel |
| D | Go to dashboard |
| R | Open reports |
| A | Show alerts |
| M | Messages |
| S | Settings |
| ? | Show help/shortcuts |

## Patient Search

| Shortcut | Action |
|----------|--------|
| Ctrl+F5 | Advanced search |
| Ctrl+Shift+F | Global search |

## Common Actions

| Shortcut | Action |
|----------|--------|
| Escape | Close dialog/modal |
| Enter | Submit form |
| Ctrl+S | Save (when available) |
| Ctrl+Z | Undo (text only) |
| Ctrl+Y | Redo (text only) |
| Tab | Move to next field |
| Shift+Tab | Move to previous field |

## Pro Tips for Efficiency

### Dashboard Mastery
- **Customize dashboard**: Drag/drop widgets to rearrange
- **Save views**: Create different dashboard views for different workflows
- **Set defaults**: Platform remembers your last view
- **Use filters**: Filter widgets to show only relevant data
- **Pin favorites**: Pin frequently used filters to dashboard

### Patient Search Optimization
- **Use MRN**: Exact match, fastest search
- **Try partial name**: "Sm" matches Smith, Smythe, etc.
- **Search by DOB**: If name spelling uncertain
- **Search by phone**: Complete, unusual names easier by phone
- **Use filters**: Filter by status, location, or other attributes

### Navigation Speed
- **Use search bar**: "/" jumps to search, faster than mouse clicks
- **Keyboard navigation**: Tab through forms instead of clicking
- **Recent patients**: System shows recent patients for quick access
- **Favorite patients**: Star frequently visited patients
- **Saved filters**: Create and reuse filter combinations

### Data Entry Tips
- **Use standard abbreviations**: System recognizes common medical abbreviations
- **Copy/paste**: Works for most fields
- **Tab between fields**: Faster than mouse clicking
- **Use date picker**: Click calendar icon for easier date selection
- **Auto-complete**: Many fields offer suggestions

### Report Generation
- **Save reports**: Reuse same settings multiple times
- **Schedule reports**: Automatic generation saves time
- **Export templates**: Predefined export formats available
- **Batch operations**: Select multiple items and export together
- **Quick filters**: Pre-built common filters available

### Message and Collaboration
- **Type @ symbol**: @mention colleague to notify
- **Starred messages**: Mark important conversations
- **Read receipt**: Verify message delivered
- **Search messages**: Find old conversations
- **Templates**: Create message templates for common responses

### Mobile Tips
- **Offline mode**: Sync data before losing internet
- **Swipe navigation**: Swipe left/right to navigate
- **Pinch to zoom**: Enlarge text if too small
- **Rotate screen**: Landscape mode shows more info
- **Shortcuts work**: Mobile app supports same shortcuts

## Advanced Workflows

### Batch Operations
1. Select multiple patients using checkboxes
2. Perform action on all at once (assign, export, message)
3. Confirm action applies to batch
4. Use filters to identify patients first

### Custom Views
1. Set filters, sorting, and display preferences
2. Save as custom view
3. System remembers and loads same view next time
4. Share view with colleagues if permitted

### Workflow Automation
- **Rules-based automation**: Create rules (if X then Y)
- **Scheduled tasks**: Schedule recurring actions
- **Alerts trigger actions**: Some alerts auto-trigger workflows
- **Notification workflows**: Notifications can trigger tasks

## Accessibility Features

### Keyboard Navigation
- **Tab key**: Navigate through all interactive elements
- **Enter/Space**: Activate buttons
- **Arrow keys**: Navigate lists and menus
- **Escape**: Close modals and dropdowns

### Screen Reader Support
- **ARIA labels**: All buttons properly labeled
- **Heading structure**: Proper H1-H3 structure for navigation
- **Alt text**: Images have descriptive alt text
- **Form labels**: All fields clearly labeled

### Visual Aids
- **High contrast mode**: Available in settings
- **Font size**: Adjustable in browser settings
- **Dark mode**: Available for reduced eye strain
- **Focus indicators**: Clear focus outline on keyboard navigation

## Hidden Features

**Patient Timeline**: Click timeline icon to see chronological patient events

**Bulk messaging**: Select multiple patients and message all at once

**Export to EHR**: Some systems allow export of notes back to EHR

**Patient portal access**: View what patient sees in their portal

**Trend analysis**: Click on charts to drill into trends

**Peer benchmarking**: Compare your metrics with similar providers

## Performance Tips

- **Use filters**: Narrower datasets load faster
- **Close tabs**: Each tab uses browser memory
- **Clear cache**: Weekly cache clearing improves speed
- **Use WiFi**: Faster and more stable than cellular
- **Avoid bulk exports**: Export in smaller batches

## Common Mistakes to Avoid

❌ Typing full name when MRN faster
❌ Not saving before leaving page
❌ Double-clicking on buttons (creates duplicates)
❌ Not using filters (slow, overwhelming)
❌ Opening too many tabs (memory issue)
❌ Ignoring unsaved changes warning
