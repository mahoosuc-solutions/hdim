## KEYBOARD SHORTCUTS

| Shortcut | Action |
|----------|--------|
| Ctrl+R | Refresh page |
| / | Jump to search |
| Ctrl+F | Find on page |
| P | Patient panel |
| D | Dashboard |
| R | Reports |
| Escape | Close dialog |
| Tab | Next field |
| Enter | Submit/Search |

## NAVIGATION

1. **Patient Panel**: Click "Patients" or press P
2. **Dashboard**: Click "Dashboard" or press D
3. **Search**: Press "/" or click search box
4. **Settings**: Click your name > Settings
5. **Help**: Click "?" or "Help" button

## COMMON TASKS - QUICK STEPS

### Find a Patient
1. Press "/" or click search
2. Type name, MRN, or DOB
3. Press Enter or click patient
4. View health overview

### Create an Order
1. Click patient name
2. Go to "Orders" tab
3. Click "New Order"
4. Select type (lab, imaging, etc.)
5. Complete details
6. Submit

### Close a Care Gap
1. Go to patient's care gaps
2. Click gap to expand details
3. Select action (address, refer, decline)
4. Document what you did
5. Click "Close"

### Generate a Report
1. Click "Reports" or press R
2. Select report type
3. Set date range and filters
4. Click "Generate"
5. Export or schedule

### Send a Message
1. Click message icon
2. Search for person
3. Type message
4. Click "Send"
