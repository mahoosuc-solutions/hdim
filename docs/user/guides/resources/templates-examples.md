# Templates & Examples

Ready-to-use templates and examples for common tasks.

## Clinical Documentation Templates

### Office Visit Note Template
```
CHIEF COMPLAINT: [Patient's main reason for visit]

HISTORY OF PRESENT ILLNESS:
[Describe current symptoms, duration, severity, and context]

PAST MEDICAL HISTORY: [Relevant chronic conditions]

MEDICATIONS: [Current medications with dosages]

ALLERGIES: [Any allergies - list all]

PHYSICAL EXAMINATION:
- Vital Signs: BP [___], HR [___], RR [___], Temp [___]
- General: [appearance, alertness]
- Cardiovascular: [heart sounds, peripheral pulses]
- Respiratory: [breath sounds, respiratory effort]
- Other: [relevant findings]

ASSESSMENT: [Clinical impression and diagnosis]

PLAN:
- Medications: [prescribed medications]
- Labs: [ordered labs]
- Referrals: [specialist referrals if needed]
- Patient education: [topics discussed]
- Follow-up: [when and with whom]

PROVIDER: [Your name and credentials]
```

### Progress Note Template
```
DATE: [Date of note]
SUBJECTIVE: [What patient reports]
OBJECTIVE: [Exam findings, vital signs, lab values]
ASSESSMENT: [Clinical impression]
PLAN: [Treatment plan going forward]
```

### Phone Call Documentation
```
Call Purpose: [Why did you call or why did they call]
Patient Status: [How did patient seem - alert, concerned, etc.]
Topics Discussed: [Main points covered]
Action Items: [What needs to happen next]
Follow-up: [When/how to follow up]
Provider Notes: [Any clinical concerns noted]
```

## Care Plan Templates

### Care Plan Goal Template
```
GOAL: [Patient's main health goal - specific and measurable]

RATIONALE: [Why this goal - clinical basis]

TARGET: [Specific, measurable outcome]

TIMELINE: [When should goal be achieved]

INTERVENTIONS:
- [Specific action/intervention]
- [Assigned to: person/role]
- [Frequency: how often]

SUCCESS MEASURE: [How will we know goal achieved]

REVIEW DATE: [When to reassess]
```

### Barrier Assessment Template
```
IDENTIFIED BARRIER: [What's preventing progress]

TYPE:
☐ Medical/clinical
☐ Behavioral/habit
☐ Social/family
☐ Financial
☐ Knowledge/education
☐ Access/transportation
☐ Motivation/beliefs

CONTRIBUTING FACTORS: [Why is this a barrier]

MITIGATION STRATEGIES: [How to address]

RESPONSIBLE PERSON: [Who will help address]

TIMELINE: [When to address]
```

## Message Templates

### Patient Education Message
```
Hello [Patient Name],

I'm reaching out to make sure you understand your [condition] and how to manage it well.

Key Points:
- [Important point 1]
- [Important point 2]
- [Important point 3]

Action Items:
- [What patient should do]
- [When to do it]

If you have questions or concerns, please [contact method].

Best regards,
[Your name]
```

### Care Gap Closure Notification
```
Hello [Patient Name],

Great news! You're up-to-date on an important health screening: [care gap description].

What this means: [Explanation in plain language]

Next steps: [If any follow-up needed]

Questions? Contact me at [contact info].

[Your name]
```

### Referral Coordination Message
```
Hi [Specialist Name],

I'm referring [Patient Name] to your clinic for [condition/concern].

Patient Background:
- Age: [age]
- Diagnosis: [relevant diagnoses]
- Current Medications: [relevant meds]
- Concerns: [specific concerns]

Requested Services: [What do you want specialist to evaluate/treat]

Timeline: [How urgent - routine, urgent, emergent]

Please contact patient at [phone] or [email] to schedule.

Thank you,
[Your name]
```

## Outreach Call Script

### Appointment Reminder Call
```
"Hi [Patient Name], this is [Your Name] from [Clinic].
I'm calling to remind you about your appointment with
[Provider] on [Date] at [Time]. Are you still planning to come?"

[If yes:] "Great! Just to confirm, we're located at [address]."

[If no:] "I understand. Would you like to reschedule,
or would you prefer not to come at this time?"

[Closing:] "If you have any questions before your appointment,
please call us at [phone]. Thanks!"
```

### Medication Adherence Call
```
"Hi [Patient Name], this is [Your Name] from [Clinic].
I'm calling because we noticed you may have missed picking up
your [medication name] refill. How are you doing with that?"

[If patient has barrier:] "I understand. Let's see if we can help.
What's making it difficult?"

[Problem-solve together, offer solutions]

[Closing:] "Let's make sure you get this medication.
Here's what we can do..."
```

### Health Education Call
```
"Hi [Patient Name], this is [Your Name] from [Clinic].
I wanted to touch base about your [condition].
Do you have a few minutes to talk?"

[If yes:] "Great! I wanted to make sure you understand your
condition and how to manage it. Do you have questions about..."

[Cover key points, check understanding]

[Closing:] "Does that make sense? What questions do you have?"
```

## Documentation Examples

### Example: Medication Safety Check
```
Patient: John Smith
Medications to verify:
1. Lisinopril (ACE inhibitor) - new
2. Metformin (diabetes)
3. Aspirin (blood thinner)

Safety check:
✓ Allergies: NKDA - OK to proceed
✓ Drug interactions: No significant interactions
✓ Dosing: All appropriate for patient age/weight
✓ Renal function: Lisinopril OK at current dose
✓ Duplicates: No duplicate therapies

Recommendation: Safe to prescribe Lisinopril

Provider: [Name] - [Date]
```

### Example: Care Gap Closure Documentation
```
Patient: Jane Doe
Care Gap: Colorectal cancer screening (age 55, due)
Gap Closure Date: [Date screening ordered]

Action Taken: Ordered colonoscopy screening
Provider: [Name]
Expected Completion: [Expected date]
Status: Ordered - Awaiting scheduling confirmation

Documentation: "Patient counseled on importance of
colorectal cancer screening. Coloroscopy ordered for
[date]. Patient education provided. Patient agreed."

Expected Status Update: [Date when status will change]
```

## Quality Improvement Examples

### Audit Finding Documentation
```
Finding: Care Gap - Blood pressure control

Patients Affected: 45 patients with hypertension

Root Cause: Medications not titrated to goal despite
elevated readings documented

Corrective Action:
1. Provider education on guidelines
2. Monthly BP review for at-risk patients
3. Automated alert for BPs >160/100

Implementation Date: [Date]

Responsible: [Provider/Manager]

Expected Outcome: Reduce uncontrolled HTN by 20% in 3 months
```

### Provider Performance Improvement
```
Provider: [Name]

Goal: Improve pneumococcal vaccination rates

Current Status: 65% compliance (Below 80% target)

Action Plan:
1. Identify all eligible patients (55+ or certain conditions)
2. Systematic outreach for vaccination opportunities
3. Standing order for office visits
4. Patient education about benefits

Timeline: Implement immediately, target 80% by [date]

Monitoring: Monthly review of vaccination rates

Resources Needed: [any education/tools]
```

## Patient Education Materials

### Hypertension Management Template
```
UNDERSTANDING HIGH BLOOD PRESSURE

What is it?
- Blood pressure over [___/___] consistently

Why does it matter?
- Increases risk of heart attack, stroke, kidney disease
- Often called "silent killer" - no symptoms

What can you do?
☐ Take medications as prescribed
☐ Reduce salt intake (<2,300mg/day)
☐ Exercise 30 minutes most days
☐ Maintain healthy weight
☐ Reduce stress
☐ Limit alcohol
☐ Monitor blood pressure at home

When to call:
- BP consistently >160/100
- Chest pain or shortness of breath
- Severe headache
- Vision changes
- Any new concerning symptoms

Questions? Call [number]
```
