# Mobile App Issues

This guide covers common mobile app problems and solutions.

## App Won't Launch or Crashes

### "App Crashes When Opening"

**Problem**: App closes immediately upon launch

**Solutions** (in order):
1. **Force close app**: Settings > Apps > Close
2. **Restart phone**: Power off and on
3. **Update app**: Check App Store/Play Store for updates
4. **Clear app cache**: Settings > Apps > Clear Cache
5. **Reinstall app**: Uninstall and reinstall completely
6. **Restart phone again**: Fresh start
7. **Contact IT**: If still crashing

**Prevention**:
- Keep app updated
- Restart phone periodically
- Clear cache monthly
- Avoid multiple apps open simultaneously

### "App Keeps Crashing During Use"

**Problem**: App works initially but crashes during use

**Causes**: Memory issue, specific feature error, data too large

**Solutions**:
1. **Close other apps**: Free up memory
2. **Restart phone**: Clear memory
3. **Update app**: May have bug fix
4. **Try different feature**: Identify which crashes
5. **Clear app data**: May have corrupted cache
6. **Try web version**: Workaround while fixing
7. **Contact IT**: With specific feature causing crash

## Login and Authentication Issues

### "Can't Log In on Mobile"

**Problem**: Login credentials not working

**Solutions**:
1. **Verify username/password**: Correct credentials?
2. **Verify network**: Connected to WiFi or cellular?
3. **Try website first**: Verify credentials work there
4. **Try different network**: WiFi vs cellular
5. **Clear app data**: May have cached bad login
6. **Restart phone**: Fresh session
7. **Contact IT**: If credentials verified but app won't login

### "MFA Not Working on Mobile"

**Problem**: Can't complete multi-factor authentication

**Solutions**:
1. **Verify code**: Is code typed correctly?
2. **Check time**: Time on phone correct?
3. **Try different method**: SMS instead of app
4. **Copy/paste code**: Avoid typing errors
5. **Check signal**: Cellular signal strong enough?
6. **Restart MFA app**: Authentication app may need reset
7. **Contact IT**: If MFA not working

### "Logged Out Unexpectedly"

**Problem**: Session ended, must log back in

**Causes**: Session timeout, auto-logout, security lock

**Solutions**:
1. **Log back in**: Provide credentials again
2. **Check timeout setting**: May be set to auto-logout
3. **Check network**: Connection may have dropped
4. **Clear app data**: May help with recurring logouts
5. **Update app**: May fix logout issue
6. **Contact IT**: If repeatedly happening

## Data Sync Issues

### "Mobile App Not Syncing"

**Problem**: Patient data not updating on phone

**Causes**: Network issue, sync disabled, app needs update

**Solutions**:
1. **Check network**: Connected to WiFi or cellular?
2. **Check sync setting**: Is sync enabled?
3. **Manually sync**: Refresh/pull down to sync
4. **Restart app**: Close and reopen
5. **Update app**: Check App Store/Play Store
6. **Restart phone**: Clear memory
7. **Contact IT**: If consistently not syncing

### "Offline Mode Not Working"

**Problem**: App doesn't work without internet

**Causes**: Offline mode not enabled, data not pre-cached

**Solutions**:
1. **Check offline capability**: Some features offline, some not
2. **Ensure prior sync**: Must sync before going offline
3. **Try limited features**: Basic functions may work offline
4. **Go online for updates**: Some actions require connection
5. **Contact IT**: For offline feature questions

## Performance Issues

### "App Very Slow"

**Problem**: App lagging or sluggish

**Solutions**:
1. **Close other apps**: Free up phone memory
2. **Restart phone**: Clear memory completely
3. **Check network**: WiFi usually faster than cellular
4. **Clear app cache**: May have accumulated junk
5. **Update app**: Newer version may be faster
6. **Try web version**: Compare performance
7. **Contact IT**: If significantly slower than expected

### "Large Lists Won't Load**

**Problem**: Patient list or data lists not loading

**Solutions**:
1. **Close other apps**: Free memory
2. **Try WiFi**: Faster than cellular
3. **Scroll slowly**: Load in batches
4. **Use filters**: Reduce dataset size
5. **Try web version**: Better with large data
6. **Restart app**: Clear state
7. **Contact IT**: If consistently failing

## Notification Issues

### "Not Getting Mobile Notifications"

**Problem**: Alerts and notifications not arriving

**Solutions**:
1. **Check notification setting**: Enabled for app?
2. **Check phone notification setting**: Enabled generally?
3. **Check quiet hours**: May be disabled
4. **Test notification**: Send test alert
5. **Restart app**: May restore notifications
6. **Restart phone**: Clear state
7. **Contact IT**: If should be receiving

## Data Display Issues

### "Wrong Data Showing on Mobile"

**Problem**: Mobile shows different data than web version

**Solutions**:
1. **Refresh/sync**: Pull down to refresh
2. **Restart app**: Clear cache
3. **Try web version**: Verify which is correct
4. **Wait for sync**: May be out of sync
5. **Contact IT**: If data genuinely different

### "Can't See All Information**

**Problem**: Data fields or information not visible

**Causes**: Screen size, display format, data not loaded

**Solutions**:
1. **Scroll**: May need to scroll to see more
2. **Rotate screen**: Landscape may show more
3. **Pinch to zoom**: May be too small
4. **Try web version**: Better for detailed view
5. **Update app**: Newer version may display better

## Best Practices

### Mobile App Usage
- ✅ Keep app updated
- ✅ Clear cache monthly
- ✅ Restart phone regularly
- ✅ Use WiFi when possible
- ✅ Test app before relying on it clinically

### Security
- ✅ Use strong password
- ✅ Enable multi-factor authentication
- ✅ Don't leave app open on shared device
- ✅ Log out after use
- ✅ Keep device updated

### When Traveling
- ✅ Pre-sync data before leaving WiFi
- ✅ Use offline mode when needed
- ✅ Monitor data usage
- ✅ Keep device with you
- ✅ Have web access backup
