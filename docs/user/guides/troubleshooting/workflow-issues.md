# Workflow Issues

This guide covers common workflow and task management problems.

## Missing Tasks

### "Task Not Appearing"

**Problem**: Created task doesn't show up

**Causes**: Task not saved, filtering hiding it, visibility issue

**Solutions**:
1. **Check for save confirmation**: Did save actually complete?
2. **Refresh page**: May need to reload
3. **Check filters**: May be filtering out new tasks
4. **Verify assignment**: Is task assigned to visible person?
5. **Check status**: May be in completed or archived section
6. **Clear filters**: Show all tasks
7. **Contact IT**: If task definitely should exist

### "Task Disappeared"

**Problem**: Previously visible task now gone

**Causes**: Task completed, archived, deleted, filtered

**Solutions**:
1. **Check completed section**: May have been finished
2. **Check archive**: May have been archived
3. **Refresh page**: May be sync issue
4. **Clear filters**: May be hidden by filters
5. **Check permissions**: Access may have changed
6. **Contact supervisor**: May have been reassigned
7. **Contact IT**: If task should still be there

## Status Update Issues

### "Task Status Won't Update"

**Problem**: Can't change task status

**Causes**: Permission issue, task locked, validation error

**Solutions**:
1. **Check permissions**: Can you edit this task?
2. **Try different status**: Verify specific status issue
3. **Refresh page**: Clear cached state
4. **Try again**: May be temporary issue
5. **Check task details**: All required fields filled?
6. **Contact supervisor**: May need permission change
7. **Contact IT**: If should be able to update

### "Status Change Not Saving"

**Problem**: Change status but it reverts

**Solutions**:
1. **Wait a moment**: System may be processing
2. **Refresh page**: Verify change was saved
3. **Try again**: May not have saved first time
4. **Check permissions**: Verify you can edit
5. **Try different status**: Verify specific issue
6. **Clear cache**: Cached data causing issue
7. **Contact IT**: If consistently not saving

## Task Assignment Issues

### "Can't Assign Task"

**Problem**: Assignment button not working

**Causes**: Permission issue, person not available, task locked

**Solutions**:
1. **Check permissions**: Can you reassign?
2. **Verify assignee**: Person still on team?
3. **Try different person**: Verify system works
4. **Check task status**: May be locked when completed
5. **Refresh page**: Clear cached state
6. **Contact supervisor**: May need permission
7. **Contact IT**: If should be able to assign

### "Wrong Person Assigned"

**Problem**: Task shows assigned to wrong person

**Causes**: Assignment error, sync issue, display error

**Solutions**:
1. **Reassign correctly**: Update to right person
2. **Verify assignment saved**: Confirm change
3. **Notify wrong assignee**: Let them know of error
4. **Notify correct assignee**: Alert them of reassignment
5. **Document issue**: Note timing and what happened
6. **Contact IT**: If assignment error pattern

### "Can't Unassign Task"

**Problem**: Task assigned but can't remove assignment

**Solutions**:
1. **Try reassigning**: Assign to different person
2. **Check permissions**: Do you have unassign permission?
3. **Try different task**: Verify system works generally
4. **Contact supervisor**: May need permission
5. **Reassign to supervisor**: Temporary workaround

## Workflow Automation Issues

### "Automated Task Not Running"

**Problem**: Scheduled workflow didn't execute

**Causes**: Automation disabled, condition not met, system error

**Solutions**:
1. **Check if enabled**: Is automation turned on?
2. **Verify conditions**: Are conditions actually met?
3. **Check schedule**: Is it scheduled for right time?
4. **Wait for next cycle**: May run at next scheduled time
5. **Contact IT**: If should have run
6. **Manually run**: Workaround until fixed

### "Workflow Creating Wrong Outcomes"

**Problem**: Automated workflow creates incorrect tasks/assignments

**Causes**: Workflow misconfigured, condition logic error

**Solutions**:
1. **Stop workflow**: Disable to prevent further errors
2. **Document issue**: What's happening incorrectly?
3. **Contact IT**: Need to review workflow logic
4. **Manual cleanup**: May need to fix created items
5. **Don't rely on automation**: Use manual process until fixed

## Notification Issues

### "Not Getting Task Notifications"

**Problem**: No notification when task assigned

**Causes**: Notifications disabled, settings wrong, email issue

**Solutions**:
1. **Check notification settings**: Are they on?
2. **Verify email address**: Correct in system?
3. **Check spam folder**: May be filtered
4. **Test notification**: Create test task to verify
5. **Contact IT**: If settings correct but not receiving

## Best Practices

### Task Management
- ✅ Create clear task descriptions
- ✅ Assign to right person
- ✅ Set realistic due dates
- ✅ Update status regularly
- ✅ Close tasks promptly

### Workflow Efficiency
- ✅ Use automation for routine tasks
- ✅ Review workflow settings regularly
- ✅ Monitor automation accuracy
- ✅ Escalate errors quickly
- ✅ Adjust workflows based on experience

### Communication
- ✅ Notify when assigning critical tasks
- ✅ Provide clear task context
- ✅ Follow up on overdue tasks
- ✅ Communicate changes promptly
- ✅ Document decisions
