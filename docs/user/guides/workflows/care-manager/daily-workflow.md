# Care Manager Daily Workflow

Your day as a care manager focuses on patient outreach, care coordination, and helping patients manage their health. This guide walks you through starting your day and managing your daily workflow.

## Quick Reference

| Task | Time | Priority | Frequency |
|------|------|----------|-----------|
| Review Daily Caseload | 10 min | High | Daily |
| Prioritize Outreach | 5 min | High | Daily |
| Make Patient Calls | Variable | High | Throughout day |
| Document Interactions | 2-3 min | Critical | Each contact |
| Update Care Plans | 3-5 min | Medium | As needed |
| Team Communication | 2-3 min | Medium | Daily |
| End-of-Day Summary | 5 min | Medium | Daily |

## Morning Startup (15-20 Minutes)

### Step 1: Login and System Check
1. Navigate to HealthData in Motion platform
2. Enter your credentials and complete authentication
3. Click **Sign In**
4. Verify you see your name and role (Care Manager) in top-right
5. Check for any system alerts or maintenance notices

### Step 2: Review Your Notification Dashboard
1. Click **Notifications** bell icon (top navigation)
2. Review alerts in order of priority:
   - 🔴 **Critical** (Red): Urgent patient issues
   - 🟠 **High** (Orange): Same-day action needed
   - 🟡 **Standard** (Blue): Regular workflow items
3. Note any critical items requiring immediate attention

### Step 3: Load Your Caseload
1. Click **My Patients** or **My Caseload**
2. Platform displays all patients assigned to your care
3. System shows key information at top:
   - **Total Patients**: Your full caseload size
   - **High-Risk Patients**: Count needing extra attention
   - **New Admits**: Recently assigned patients
   - **Follow-ups Due**: Patients needing contact today
   - **Task Count**: Items on your to-do list

### Step 4: Review Daily Tasks
1. Look for **Daily Tasks** section (usually top or sidebar)
2. Shows items generated overnight:
   - **New Alerts**: Patients with new risk alerts
   - **Follow-ups Due**: Patients you need to contact today
   - **Care Plan Updates**: Plans requiring review/modification
   - **Referral Updates**: Specialist responses arrived
   - **ED/Hospital Admissions**: New high-utilizers
3. Count total tasks for the day
4. Sort by priority if needed

## Identifying Your Day's Priorities (5-10 Minutes)

### Priority Tier System

**Tier 1 - Immediate (Next 1-2 Hours)**:
- 🔴 New ED/hospital admissions (needs immediate discharge planning)
- 🔴 Critical alerts (dangerous lab values, medication issues)
- 🔴 Patient safety concerns (falls, abuse, crisis)
- 🔴 Post-discharge follow-ups (within 24-48 hours)

**Tier 2 - Today (Morning/Afternoon)**:
- 🟠 High-risk patients with new care needs
- 🟠 Patients with overdue follow-ups
- 🟠 Care plan updates needed urgently
- 🟠 Medication adherence issues

**Tier 3 - This Week (Schedule Appropriately)**:
- 🟡 Routine patient check-ins
- 🟡 Preventive care planning
- 🟡 Care plan annual reviews
- 🟡 Patient education scheduling

**Tier 4 - As Time Allows**:
- 🟢 Administrative tasks
- 🟢 Documentation updates
- 🟢 Team coordination

### Filter Your Caseload for Today

Use filters to create focused view:

1. Click **Advanced Filters**
2. Set filters:
   - **Follow-ups Due**: Today only
   - **Risk Level**: Red and Orange (or customize)
   - **Care Status**: Needs intervention or review
   - **Assignment Status**: Only your patients
3. Click **Apply Filters**
4. Narrowed list shows patients you should contact today

## Core Daily Tasks

### Task 1: Make Patient Outreach Calls (30-60 minutes)

**Prepare for Calls** (5 minutes):
1. Review your filtered patient list
2. Prepare call script/talking points (3-5 patients at a time)
3. Have:
   - Patient contact information visible
   - Their care plan open
   - Recent interaction notes
   - Any new lab results or medical updates

**Making Calls** (2-5 minutes per patient):
1. Use platform's **Call Log** feature (or use phone, document after)
2. Click **Log Call** for patient
3. Document:
   - **Contact Status**: Reached patient / Left message / Wrong number / Refused
   - **Call Duration**: Approximate time
   - **Topics Discussed**: Key points covered
   - **Agreements Made**: What patient agreed to
   - **Next Steps**: Follow-up plan
4. Save call documentation
5. Create follow-up task if needed

**Call Content** (varies by patient):
- Ask how they're doing (opening)
- Check medication compliance
- Discuss recent doctor visit/lab results
- Identify barriers to care
- Make referrals if needed
- Schedule appointments
- Provide education/encouragement
- Close with clear next steps

**Documentation After Calls**:
- Call logs auto-save if using platform phone feature
- If using personal phone, manually log call in platform within 1 hour
- Include:
  - What was discussed
  - Agreements made
  - Any concerns identified
  - When you'll follow up

### Task 2: Review New Alerts and Events

Throughout your day:
1. Check **Notifications** regularly (every 30-60 minutes)
2. New alerts appear as they're generated:
   - **Hospital Admission**: Immediate action needed
   - **ED Discharge**: Follow-up within 24 hours
   - **Lab Result Alert**: If abnormal
   - **Medication Alert**: Interaction or adherence issue
   - **Risk Score Change**: Patient became high-risk
3. Click alert to see details
4. Determine action needed:
   - Call patient immediately?
   - Assign to care plan?
   - Refer to physician?
   - Document and monitor?
5. Take action and document

### Task 3: Update Care Plans as Needed

When patient circumstances change:
1. Click patient name to open detail page
2. Find **Active Care Plan** section
3. Review current plan goals/interventions
4. Determine if update needed:
   - Patient reached goal? (update)
   - New condition identified? (add goal)
   - Barrier identified? (modify intervention)
   - Medication change? (update plan)
5. Click **Edit Care Plan**
6. Make necessary changes:
   - Add new goals
   - Update target dates
   - Modify interventions
   - Document reason for change
7. Save changes
8. Communicate update to care team (see collaboration section)

## Collaboration and Communication

### Reaching Your Team

**Communication Methods**:
1. **Secure Messaging**: For non-urgent coordination
   - Click patient name
   - Find **Care Team Messages** section
   - Send message to physician, RN, social worker
   - Get notifications when team responds
2. **Task Assignments**: For action-oriented items
   - Create task and assign to team member
   - Include due date and details
   - Assignee gets notification
   - Can track completion
3. **Care Conferences**: For complex patients
   - Scheduled team meetings
   - Discuss patient progress
   - Align on care plan
   - Document decisions

### Daily Team Huddle

If your organization has daily huddles:
1. **Before Huddle** (5 minutes): Prepare updates on:
   - New ED/hospital admissions
   - High-risk patient changes
   - Medication or care plan updates
   - Barriers you've identified
2. **During Huddle**: Share key patient info
3. **After Huddle**: Update care plans based on decisions

### Escalation Path

For issues needing urgent attention:
1. **Minor Issues**: Message assigned physician
2. **Moderate Issues**: Call physician directly
3. **Urgent Safety Issues**: Call physician immediately or page
4. **Crises**: Follow organizational crisis protocol

## Documentation Throughout the Day

### Real-Time Documentation
Best practice: Document as you go rather than batch at end of day

**Types of Documentation**:
1. **Call Logs**: Contact attempts and conversations
2. **Care Plan Notes**: Updates to patient goals/interventions
3. **Task Documentation**: Actions taken on tasks
4. **Message Documentation**: Important team conversations
5. **Event Documentation**: Significant patient events/changes

**How to Document**:
1. After patient contact, immediately click **Add Note**
2. Select note type (Call, Visit, Message, etc.)
3. Complete required fields:
   - Date/time of interaction
   - Type of interaction
   - Content/summary
   - Patient response
   - Next steps
4. Save note immediately
5. Attach to appropriate care plan if relevant

**Why Real-Time Matters**:
- Ensures accurate details (forget details if wait)
- Prevents documentation pile-up at end of day
- Alerts team to issues immediately
- Complies with many care documentation standards

## Managing Your Task List

### Daily Task Organization

1. From **My Tasks**, filter by:
   - **Due Date**: Today
   - **Priority**: High
   - **Status**: Open
2. View filtered task list
3. For each task:
   - Click task to see full details
   - Complete the action
   - Document what you did
   - Mark task **Completed**
4. Check **Completed Today** count at end of day

### Creating New Tasks

When you identify something needing follow-up:
1. Click **New Task**
2. Fill in:
   - **Task Name**: Clear, specific description
   - **Patient**: Who it's for
   - **Due Date**: When needed
   - **Priority**: Urgent/High/Medium/Low
   - **Assigned To**: You or team member
   - **Details**: What specifically needs to be done
3. **Save Task**
4. If assigned to team member, they're notified

## Handling Common Daily Situations

### Scenario 1: Patient Answers Phone
1. **Greeting**: "Hi [Name], this is [You] calling from [Organization]. Do you have a few minutes to talk?"
2. **Reason**: "I'm calling to check in and see how you're doing"
3. **Questions**: Ask about:
   - Recent doctor visit (if one occurred)
   - Medication compliance
   - Any health changes/concerns
   - Barriers to care
4. **Actions**: Based on responses:
   - Schedule appointment
   - Arrange referral
   - Clarify medication use
   - Provide education
5. **Close**: "Thanks for talking with me. Let's check back in [timeframe]. Here's my number if you need anything before then."

### Scenario 2: Patient Doesn't Answer - Leave Voicemail
1. "Hi [Name], this is [You] from [Organization]. I'm checking in to see how you're doing and to follow up on [specific reason]. Can you please call me back at [number]? If I don't hear from you by [date], I'll try you again."
2. Document as "Left message"
3. Plan follow-up attempt in 2-3 days

### Scenario 3: New Hospital Admission Alert
1. **Immediate**: Click alert to see admission details
2. **Information Gathering**:
   - Why was patient admitted?
   - What's the likely discharge date?
   - Who's the hospital case manager?
3. **Action**:
   - Call hospital to introduce yourself
   - Understand discharge plan
   - Coordinate post-discharge follow-up
   - Plan for any equipment/support needed
4. **Documentation**:
   - Create high-priority follow-up task
   - Add to care plan if needed
   - Alert physician
5. **Follow-up**: Call patient within 24-48 hours of discharge

## End-of-Day Routine (5-10 Minutes)

### Before You Log Off

1. **Review Incomplete Documentation**:
   - Check for any calls/interactions not yet documented
   - Complete any in-progress notes
   - Don't leave work with pending documentation

2. **Check Message Queue**:
   - Any responses from team you need to act on?
   - Any patient questions needing follow-up?
   - Reply as appropriate

3. **Review Tomorrow's Tasks**:
   - Scan **Tomorrow's Tasks** section
   - Note any special needs or complex patients
   - Prepare your materials if possible
   - Update your mental checklist for morning

4. **Complete Care Team Handoff** (if applicable):
   - If leaving shift, brief next care manager
   - Highlight urgent items
   - Hand off any in-progress patients

5. **Verify All Saved**:
   - All documentation completed
   - All notes saved
   - All tasks updated
   - All messages addressed

6. **Logout Securely**:
   - Click your name in top-right
   - Select **Logout**
   - Close browser tab
   - On shared workstations, clear cache if prompted

## Efficiency Tips

### Time Management
1. **Batch similar tasks**: Group calls together, do documentation together
2. **Use templates**: For common messages/documentation
3. **Prioritize ruthlessly**: Do Tier 1 before Tier 3
4. **Use filters**: Show only relevant patients
5. **Keyboard shortcuts**: Learn platform shortcuts (press **?** to see them)

### Technology Tips
1. **Call logging**: Use platform phone feature if available (auto-logs)
2. **Templates**: Save frequently used messages/care plan language
3. **Favorites**: Star high-priority patients for quick access
4. **Notifications**: Set up custom alerts for critical items
5. **Search**: Learn advanced search for finding specific patients

### Team Collaboration
1. **Clear communication**: Be specific in messages to team
2. **Don't repeat**: Check if someone already contacted patient
3. **Share knowledge**: Document what you learn about barriers
4. **Support peers**: Help newer care managers
5. **Continuous learning**: Ask questions about best practices

## Troubleshooting Common Issues

### "Can't Find a Patient in My Caseload"
**Solution**:
1. Check current filters (might be hidden by filter)
2. Use **Search** to find patient
3. Verify patient is assigned to you
4. Contact your supervisor if assignment unclear

### "Documentation Not Saving"
**Solution**:
1. Check required fields (marked with *)
2. Verify internet connection is stable
3. Try different browser
4. Contact IT if error persists

### "Can't Reach Patient - Phone Number Seems Wrong"
**Solution**:
1. Check patient record for multiple phone numbers
2. Review recent documentation (last person to call)
3. Try other numbers if available
4. Try calling patient's emergency contact
5. Document attempts carefully
