# Registered Nurse Guide

This guide is designed for **Registered Nurses** using the HDIM Clinical Portal to coordinate patient care and manage quality improvement initiatives.

## Overview

As an RN, you are responsible for:
- Coordinating patient outreach for care gaps
- Managing care gap workflows
- Supporting providers with quality initiatives
- Monitoring population health metrics
- Documenting interventions and outcomes

---

## Your Dashboard

When you log in, you'll see the **RN Dashboard** with these widgets:

### Outreach Tasks Widget
Shows patients requiring outreach:
- Patient name and contact info
- Gap type and urgency
- Last contact attempt
- Preferred contact method

**Quick Actions:**
- Click patient → View details
- Click "Log Attempt" → Record outreach
- Click "Schedule" → Create follow-up

### Care Gap Summary Widget
Overview of population care gaps:
- Total open gaps
- By urgency level
- By gap type
- Trend vs previous period

### Recent Interventions Widget
Shows recently documented interventions:
- Patient name
- Intervention type
- Outcome
- Date/time

### Compliance Trends Widget
Population compliance metrics:
- Overall compliance rate
- Trend chart
- Top improving measures
- Measures needing attention

## Daily Workflow

### Morning Review (15-20 minutes)

1. **Review Outreach Queue**
   - Check "Outreach Tasks" widget
   - Prioritize by urgency and due date
   - Note patients with multiple gaps

2. **Check Pending Follow-ups**
   - Review scheduled follow-ups
   - Note any overdue items
   - Plan day's outreach activities

3. **Review Population Metrics**
   - Check "Compliance Trends" widget
   - Note any concerning trends
   - Identify focus areas

### Outreach Activities

For each outreach task:

1. **Prepare Patient Information**
   - Review patient demographics
   - Check contact preferences
   - Review gap details and history

2. **Conduct Outreach**
   - Call at optimal times (9-11 AM, 5-7 PM)
   - Follow approved scripts
   - Verify patient identity
   - Explain purpose and recommended action

3. **Document Outcome**
   - Log attempt immediately
   - Record outcome:
     - Reached/Left Message/No Answer
     - Appointment Scheduled
     - Patient Declined
   - Add detailed notes

4. **Schedule Follow-up**
   - If not reached: 48-hour follow-up
   - If pending action: appropriate timeline
   - If escalation needed: notify supervisor

## Managing Outreach

### View Outreach Queue

1. Navigate to Dashboard
2. Locate "Outreach Tasks" widget
3. Or navigate to **Care Gaps** → Filter "Outreach Needed"

### Log Outreach Attempt

1. Select patient/task
2. Click **"Record Intervention"**
3. Select type: Phone Call, Email, Letter, etc.
4. Select outcome:

| Outcome | Description |
|---------|-------------|
| **Reached** | Spoke with patient directly |
| **Left Message** | Left voicemail |
| **No Answer** | No response, no voicemail available |
| **Wrong Number** | Contact info incorrect |
| **Appointment Scheduled** | Visit booked |
| **Patient Declined** | Patient refused intervention |

5. Add notes with details
6. Save intervention

### Schedule Follow-up

1. Click **"Create Follow-up"**
2. Set follow-up date
3. Assign to self or colleague
4. Add context notes
5. Save

### Outreach Best Practices

**Optimal Call Times:**
| Time | Success Rate |
|------|-------------|
| 9-11 AM | High |
| 12-2 PM | Low (lunch) |
| 2-5 PM | Medium |
| 5-7 PM | High (after work) |

**Communication Tips:**
- Be clear and concise
- Respect patient's time
- Acknowledge barriers
- Offer scheduling assistance
- Represent organization professionally

## Care Gap Management

### View All Care Gaps

1. Navigate to **Care Gaps**
2. Use filters:
   - Urgency (High/Medium/Low)
   - Type (Screening/Lab/Medication/etc.)
   - Patient (search by name/MRN)
   - Status (Open/Pending/Closed)

### Close Care Gap

After successful intervention:

1. Select the gap
2. Click **"Close Gap"**
3. Select reason:
   - **Completed** - Action taken
   - **Not Applicable** - Patient excluded
   - **Patient Declined** - Patient refused
4. Add closure notes
5. Reference related documentation
6. Submit

### Bulk Operations

For efficiency with multiple gaps:

1. Select multiple gaps (checkboxes)
2. Click **"Bulk Close"** or **"Bulk Assign"**
3. Apply common reason/assignment
4. Confirm action

## Running Evaluations

### Single Patient Evaluation

1. Navigate to **Evaluations**
2. Search and select patient
3. Select quality measure
4. Click **"Evaluate"**
5. Review and act on result

### Batch Evaluation

For population assessment:

1. Navigate to **Patients**
2. Filter to target population
3. Select patients
4. Click **"Batch Evaluation"**
5. Select measure(s)
6. Run and monitor progress
7. Review results and create outreach tasks

---

## Population Health

### Identify High-Risk Patients

1. Navigate to **Patients**
2. Sort by compliance rate (ascending)
3. Or filter by number of care gaps
4. Focus outreach on highest-need patients

### Track Compliance Trends

1. Navigate to **Reports**
2. Generate Population Report
3. Select reporting period
4. Review trends by measure
5. Identify improvement opportunities

### Campaign Management

For organized outreach:

1. Identify target population
2. Create patient list
3. Assign outreach tasks
4. Track progress
5. Report outcomes

## Coordination with Providers

### Pre-Visit Preparation

Support providers by:
1. Reviewing patient care gaps before visits
2. Noting items to address during visit
3. Preparing patient education materials
4. Alerting provider to urgent items

### Post-Visit Follow-up

After patient visits:
1. Verify gaps were addressed
2. Follow up on pending orders
3. Schedule any needed follow-up appointments
4. Close gaps as completed

### Escalation

When to escalate to provider:
- Complex clinical decisions
- Patient questions beyond scope
- Adverse events or concerns
- Requests for medication changes

## Documentation Best Practices

### Intervention Notes

Include:
- Date and time of contact
- Method of contact
- Person reached (patient, family, etc.)
- Discussion summary
- Outcome and next steps

### Gap Closure Notes

Include:
- Action taken
- Date action completed
- Reference to documentation (order #, visit date)
- Any follow-up needed

## Tips for Success

### Maximize Outreach Success

1. **Use preferred contact method** - Check patient preferences
2. **Call at optimal times** - Avoid lunch, try early morning or evening
3. **Be prepared** - Know gap details before calling
4. **Document immediately** - Capture details while fresh

### Improve Population Health

1. **Prioritize high-urgency gaps** - Address critical items first
2. **Batch similar tasks** - Group outreach by gap type
3. **Track patterns** - Note common barriers
4. **Collaborate with team** - Share what works

### Common Issues

| Issue | Solution |
|-------|----------|
| Can't reach patient | Try different times, methods |
| Patient angry/frustrated | Acknowledge, escalate if needed |
| Wrong contact info | Flag for update, try alternative |
| Gap not closing | Verify all requirements met |
