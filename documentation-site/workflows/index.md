# Clinical Workflows

This section documents the key clinical workflows in the HDIM Clinical Portal. Each workflow includes step-by-step instructions, expected outcomes, and troubleshooting tips.

## Workflow Categories

| Category | Description |
|----------|-------------|
| [Daily Workflows](/workflows/provider-daily) | Start-of-day and end-of-day routines |
| [Care Gap Closure](/workflows/care-gap-closure) | Identifying and closing care gaps |
| [Quality Evaluation](/workflows/quality-evaluation) | Running quality measure evaluations |
| [Patient Outreach](/workflows/patient-outreach) | Coordinating patient communication |
| [Report Generation](/workflows/report-generation) | Creating compliance reports |

## Workflow Legend

Each workflow document includes:

- **Participants**: Roles involved in the workflow
- **Trigger**: What initiates the workflow
- **Steps**: Numbered actions to complete
- **Expected Outcomes**: What success looks like
- **Time Estimate**: Typical duration
- **Related User Stories**: Links to relevant user stories

## Quick Reference

### Most Common Workflows

1. **[Run a Quality Evaluation](/workflows/quality-evaluation)**
   - Select patient → Select measure → Submit → View result
   - Time: 2-5 minutes

2. **[Close a Care Gap](/workflows/care-gap-closure)**
   - Review gap → Document intervention → Close with reason
   - Time: 3-10 minutes

3. **[Generate a Report](/workflows/report-generation)**
   - Select type → Configure → Generate → Export
   - Time: 1-2 minutes
