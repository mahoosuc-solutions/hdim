🎯 On-Screen Help System

Comprehensive contextual help for HealthData In Motion Clinical Portal

📖 Overview

The help system provides three levels of assistance to guide users through the application:

  • Inline Tooltips - Quick contextual help next to form fields and buttons
  • Help Panel - Detailed documentation with searchable sections
  • Help Content Service - Centralized content management system

1️⃣ Help Tooltip Component

Inline contextual help that appears on hover or click. Perfect for form field clarification.

💡 Hover over the "?" icons above to see tooltips in action!

2️⃣ Help Panel Component

Comprehensive help documentation in a sliding side panel with search, sections, and quick links.

💻 Implementation Example

Add help to your page in just a few lines of code:

// Import components import { HelpTooltipComponent } from './shared/components/help-tooltip'; import { HelpPanelComponent } from './shared/components/help-panel'; import { HelpContentService } from './services/help-content.service'; // In your component @Component({ imports: [HelpTooltipComponent, HelpPanelComponent] }) export class YourComponent { showHelp = false; helpSections = this.helpContent.getMeasureBuilderHelp(); constructor(private helpContent: HelpContentService) {} } // In your template <app-help-tooltip text="Helpful tip here"></app-help-tooltip> <app-help-panel [isOpen]="showHelp" [sections]="helpSections" (close)="showHelp = false"> </app-help-panel>
