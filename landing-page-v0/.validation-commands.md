# Landing Page Validation - Quick Reference

## Installation

```bash
npm install
```

## Commands

### Run All Validations
```bash
npm run validate:all
```

### Individual Validations

```bash
npm run validate:content    # Check for placeholders, missing content
npm run validate:links      # Check external links
npm run test:e2e            # Run E2E tests (all browsers)
npm run test:e2e:ui         # Run E2E tests with UI (debugging)
npm run test:lighthouse     # Run Lighthouse performance audit
```

### Development

```bash
npm run dev                           # Start dev server
BASE_URL=http://localhost:3000 npm run test:e2e  # Test against local
```
