## Section 5: On-Page SEO Guidelines

### 5.1 Title Tags

**Format:** Primary Keyword | Secondary Keyword | HDIM
**Length:** 50-60 characters

**Examples:**
- HEDIS Software for Small Practices | Real-Time Quality | HDIM
- MIPS Reporting Made Simple | Healthcare Quality | HDIM

### 5.2 Meta Descriptions

**Length:** 150-160 characters
**Include:** Primary keyword, value proposition, CTA

**Examples:**
- Real-time HEDIS quality measurement for practices of all sizes. 61 measures, 2-week setup, starting at $49/mo. Request a demo today.

### 5.3 Header Structure

```
H1: Primary keyword + value prop (1 per page)
  H2: Section headers (4-8 per page)
    H3: Subsection headers
      H4: Detail headers (as needed)
```

### 5.4 Content Guidelines

- **Word count:** 1,500-3,000 words for pillar content
- **Keyword density:** 1-2% (natural usage)
- **Internal links:** 3-5 per article
- **External links:** 1-3 authoritative sources
- **Images:** 2-5 with alt text containing keywords
- **Lists/tables:** Use for scannability

### 5.5 Technical SEO

- **URL structure:** /blog/primary-keyword-phrase
- **Canonical tags:** Self-referencing
- **Schema markup:** Article, FAQ, How-to as applicable
- **Page speed:** Target <3 seconds
- **Mobile:** Fully responsive
