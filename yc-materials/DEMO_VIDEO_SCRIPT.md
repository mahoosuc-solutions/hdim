## Voiceover Tips

### Tone
- **Confident but not salesy**
- **Personal when discussing mission**
- **Technical but accessible**
- **Urgent but not desperate**

### Pacing
- Slightly slower than conversational
- Pause briefly at section transitions
- Let demos breathe (don't talk over every click)

### Key Phrases to Emphasize
- "Real-time" (vs. batch)
- "Under 200 milliseconds"
- "During the visit"
- "Every gap closed is a life potentially saved"
- "$80 a month" (cost contrast)

## Recording Checklist

### Before Recording
- [ ] Demo environment running and stable
- [ ] Patient data pre-loaded
- [ ] Browser profile clean
- [ ] Screen recording software configured
- [ ] Microphone tested
- [ ] Script printed/accessible
- [ ] Water nearby (for voiceover)

### During Recording
- [ ] Record at 1080p
- [ ] Smooth, deliberate mouse movements
- [ ] Pause between sections for editing
- [ ] Do multiple takes of each section
- [ ] Watch for loading delays or errors

### After Recording
- [ ] Review footage for quality
- [ ] Trim dead air and mistakes
- [ ] Add timer overlay for measure calculation
- [ ] Add transitions between sections
- [ ] Add background music
- [ ] Export at 1080p, H.264
- [ ] Keep under 100MB for upload
